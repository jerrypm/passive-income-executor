## Quick Start

### 1. Generate your Nostr identity

```bash
python3 generate_keypair.py
```

This creates a new keypair and saves it to `.env`. Your keys:
- `NOSTR_PRIVKEY` — hex private key (keep secret!)
- `NOSTR_PUBKEY` — hex public key
- `NOSTR_NSEC` — bech32 private key
- `NOSTR_NPUB` — bech32 public key (share this)

### 2. Post your first message

```bash
python3 post_to_nostr.py "Hello Nostr! My first post from the terminal."
```

### 3. Set up your AI DVM

Make sure Ollama is running with at least one model:

```bash
# Install Ollama: https://ollama.com
ollama pull llama3
```

Then start the DVM:

```bash
python3 dvm_text_generation.py
```

Your DVM will listen for kind 5050 requests on Nostr relays and respond with AI-generated text.

### 4. Announce your DVM

```bash
python3 announce_dvm.py
```

### 5. Set up automated posting

```bash
# Test content generation
python3 auto_content.py --dry-run

# Post for real
python3 auto_content.py

# Schedule via cron (every 8 hours)
crontab -e
# Add: 0 */8 * * * cd /path/to/nostr-ai-toolkit && python3 auto_content.py
```

### 6. Publish a long-form article

```bash
python3 publish_longform.py \
  --title "How I Built an AI Bot on Nostr" \
  --file article.md \
  --tags "nostr,ai,python"
```

### 7. List products on Shopstr

Edit `list_product_shopstr.py` with your own products, then:

```bash
python3 list_product_shopstr.py
```

### 8. Update your profile

Edit `update_profile.py` with your name, bio, and Lightning address, then:

```bash
python3 update_profile.py
```

## Configuration

Create a `.env` file in the same directory as the scripts:

```env
# Nostr Keys (generated by generate_keypair.py)
NOSTR_PRIVKEY=your_hex_private_key
NOSTR_PUBKEY=your_hex_public_key
NOSTR_NSEC=nsec1...
NOSTR_NPUB=npub1...

# Lightning (optional, for receiving zaps)
LIGHTNING_ADDRESS=you@walletofsatoshi.com
```

## FAQ

**Do I need to register anywhere?**
No. Nostr is permissionless. Generate a keypair and you're live.

**Is Ollama really free?**
Yes. Ollama runs AI models locally on your hardware. No API keys, no usage fees.

**Which relays does this use?**
By default: `wss://nos.lol` and `wss://relay.damus.io`. You can add more in each script.

**Can I customize the DVM model?**
Yes. Change `DEFAULT_MODEL` in `dvm_text_generation.py` to any model you have in Ollama.
