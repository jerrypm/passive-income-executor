## CATEGORY 1: DIRECTORY & PROGRAMMATIC SEO WEBSITES

### 1.1 Niche Directory Website (Programmatic SEO)
- **Concept:** Build a curated directory for a specific niche (AI tools, remote jobs, local services, events, thrift stores, etc.) using programmatic SEO to auto-generate thousands of pages targeting long-tail keywords.
- **Revenue:** $1,200-$3,000/month passively (real example: one portfolio owner earns $2,500-$3,000/month from multiple directories)
- **Monetization:** Paid listings ($50-200/month), featured placements, affiliate commissions, lead generation
- **Why passive:** Once pages are indexed and ranking, traffic compounds. SEO-driven traffic is self-sustaining.
- **Tech:** Next.js/Astro + database + programmatic page generation
- **Sources:** Outscraper case study, Directorist, MakerThrive
- **Real example:** First directory in thrifting space reached 1,000 visitors/day in 6 months, $1,200 first month

### 1.2 AI Tools Directory
- **Concept:** Curated directory of AI tools organized by category, with reviews, comparisons, and affiliate links
- **Revenue:** High affiliate commissions (20-40% recurring from SaaS tools)
- **Monetization:** Affiliate commissions + paid featured listings from AI tool creators
- **Why passive:** AI market is $2.52 trillion; new tools launch daily, directory stays relevant
- **Real examples:** TheresAnAIForThat, AIXploria, ProductHunt
- **Sources:** Directorist, various programmatic SEO guides

### 1.3 Remote Job Board
- **Concept:** Niche job board for a specific tech stack or industry (e.g., "Swift Jobs", "Rust Jobs", "Web3 Jobs")
- **Revenue:** Industry worth $14.7B; successful boards earn $2,000-$10,000/month
- **Monetization:** Employer job posting fees ($99-299/post), resume database access, featured listings
- **Why passive:** Employers expect to pay for job postings; once traffic/community exists, it self-sustains
- **Tech:** JBoard, custom Next.js, or WordPress
- **Sources:** JBoard, Job Boardly, Converjit
- **Critical:** Need 10,000+ monthly visitors for financial viability

### 1.4 Events & Learning Directory
- **Concept:** Directory of workshops, bootcamps, webinars, certifications in a specific field
- **Revenue:** Booking commissions, course affiliate commissions, sponsored listings
- **Why passive:** Skill improvement is non-negotiable; professionals constantly search for events
- **Sources:** Directorist, KidzyClub example

## CATEGORY 5: DIGITAL PRODUCTS & TEMPLATES

### 5.1 Notion Templates
- **Concept:** Create productivity templates, project trackers, client CRMs for specific niches
- **Revenue:** Beginners $500-$1,500/month within 6 months; established $3,000-$8,000/month; top creators $200K+/year
- **Pricing:** $19-$39 sweet spot; premium bundles $150-$300+
- **Why passive:** No inventory, sell forever once created; AI makes creation faster
- **Platforms:** Gumroad, Etsy, own website, Notion marketplace
- **Real examples:** Thomas Frank earned $1M+; Easlo earned $239K in one year
- **Sources:** Kupkaike, Stackademic, EzyCourse, Coachli

### 5.2 Framer/Webflow Templates
- **Concept:** Design and sell website templates on Framer or Webflow marketplace
- **Revenue:** $500-$2,000 per template; top creators $21,733/month (June 2025 data)
- **Framer:** Keep 100% revenue; 2,500 templates on marketplace
- **Webflow:** Keep 80% revenue; 7,000 templates; 6x more traffic (10.4M vs 1.7M visitors)
- **Why passive:** Webflow marketplace is especially passive - consistent sales without own marketing
- **Sources:** Bryn Taylor, SegmentUI, AllAboutFramer

### 5.3 Figma UI Kits & Design Systems
- **Concept:** Create UI kits, design systems, icon packs for specific niches
- **Revenue:** $500-$20,000/month
- **Why passive:** Designers always need starting points; once listed, sells perpetually
- **Real example:** One designer made $300K from Figma Academy in 4 months
- **Sources:** IdeaProof, UpMenu

### 5.4 Shopify Sections/Themes (Pre-built Code)
- **Concept:** Sell pre-built Shopify code sections and themes - code bundled with educational content
- **Revenue:** $2,221/month from Gumroad + Shopify store (real data from Eduard Fastovski)
- **Why passive:** One-time payment products; merchants prefer this over app subscriptions
- **Source:** Eduard Fastovski ($5K/month total passive income)

### 5.5 SaaS Starter Kit / Boilerplate
- **Concept:** Pre-configured tech stack (e.g., Next.js + Supabase + Stripe) sold to other developers
- **Revenue:** $100+ per kit; SaaS Pegasus generates 80% of creator's income (1,000+ users)
- **Why passive:** Sell the same codebase repeatedly; developers pay to skip setup
- **Real example:** Cory Zue's SaaS Pegasus
- **Sources:** Cory Zue, DEV Community

### 5.6 Curated AI Training Datasets
- **Concept:** Create specialized, high-quality datasets for AI model fine-tuning and sell on Hugging Face/Kaggle
- **Revenue:** Premium pricing; market growing from $4.44B (2026) to $23.18B (2034) at 22.9% CAGR
- **Why passive:** Create once, license repeatedly; demand growing exponentially
- **Example:** Cleaned dataset of 5,000 senior-level Rust code reviews
- **Sources:** DEV Community, MarketsAndMarkets, Fortune Business Insights

### 5.7 AI Prompt Libraries
- **Concept:** Curated, enterprise-grade prompt libraries for specific use cases (marketing, coding, design)
- **Revenue:** $200-$5,000/month
- **Monetization:** One-time purchase or subscription on Gumroad/own site
- **Why passive:** Businesses pay for curated prompts; create once, sell forever
- **Sources:** IdeaProof, DEV Community
