## TECHNICAL NOTES FOR APPLESCRIPT AUTOMATION

### Chrome Profile Limitation
AppleScript's `tell application "Google Chrome"` can only see windows from ONE profile at a time. Workarounds:
1. Use separate browsers for separate accounts (Chrome, Brave, Firefox, Arc)
2. Use Chrome's `--profile-directory` flag to launch specific profiles
3. For DePIN extensions: install all in the same Chrome profile (they don't conflict)

### Recommended Chrome Profile Setup
- **Profile 1 (Main):** All DePIN extensions + bandwidth sharing + cashback
- **Profile 2 (Twitter - existing):** Twitter auto-posting
- **Profile 3 (Content):** Medium, LinkedIn, Stacker News posting
- **Profile 4 (Airdrop):** DeFi interactions, daily check-ins

### AppleScript Reliability Tips
- Add random delays (30-120 seconds) between actions
- Check for page load completion before acting
- Handle error states (popup dialogs, login prompts)
- Log all actions for debugging
- Run during off-peak hours for less detection
