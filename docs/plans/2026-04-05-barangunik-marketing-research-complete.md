## 1. Shopee Affiliate Program Indonesia

### Cara Kerja
- **Portal:** https://affiliate.shopee.co.id
- **Gratis daftar**, perlu akun Shopee + social media (500-1000 followers recommended)
- **Approval:** 1-5 hari kerja
- **Cookie duration:** 7 hari (kalau klik link kita, beli APA AJA dalam 7 hari = dapat komisi)
- **Minimum payout:** Rp 100.000, via transfer bank, 5-7 hari

### Tier Affiliate
| Tier | Max Komisi/Order |
|------|-----------------|
| Shopee Affiliate (regular) | Rp 10.000 |
| Shopee Partner (higher) | Rp 50.000 |

### Komisi per Kategori

**Base commission:**
| Kategori | Rate | Cap |
|----------|------|-----|
| Non-elektronik (Fashion, Beauty, Home Decor, Toys, Aksesoris, Kitchen) | **2%** | Rp 10.000/order |
| Elektronik (HP, Laptop, Gadget) | **0.5%** | Rp 10.000/order |

**Komisi XTRA (dari seller yang opt-in AMS):**
- Tambahan **6-40% uncapped** tergantung seller
- Kategori barang unik (Home Decor, Toys, Aksesoris, Kitchen): typical XTRA **6-15%**
- **Efektif total: 8-17% per sale tanpa cap** ← INI YANG REAL MONEY-nya

### Format Affiliate Link
```
https://s.shopee.co.id/an_redir?origin_link={ENCODED_URL}&affiliate_id={ID}&sub_id={5-SEGMENT-TRACKING}
```
- Short link pakai domain `shope.ee`
- Deep link bisa langsung buka app Shopee
- Sub ID support 5 segment buat tracking multi-channel

### Tools Yang Tersedia
- Link generator, deeplink generator, short link creator
- Performance dashboard + weekly reports
- Filter produk yang punya Komisi XTRA
- **Open API** (App ID + Secret Key) di affiliate.shopee.co.id/api
- Third-party: Involve Asia, ACCESSTRADE, wecantrack (push ke Google Analytics, Zapier, dll)

| # | Format | Views | Kenapa Works |

## 4. Twitter Strategy — Bahasa Indonesia

### Format Tweet Yang Convert

**Format 1: Thread (conversion tertinggi)**
```
🧵 5 Barang Shopee Under 50rb yang Bikin Hidup Lebih Gampang

1/ Mini fan USB — cuma 23rb, bisa buat di meja kerja
[image]

Link: [shopee link]
```

**Format 2: Single tweet + image**
```
Baru tau ada barang segini bagus cuma 18rb di Shopee 😭

[product image]

Link di reply 👇
```
→ **Link di REPLY, bukan di tweet utama** (hindari penalty algoritma Twitter)

**Format 3: Video tweet (15-30 detik)**
```
Tes barang viral Shopee — mini printer portable cuma 45rb

Worth it ga? 🤔

[video clip]
```

**Format 4: Meme/relatable**
```
POV: lo baru gajian terus buka Shopee

[image cart penuh barang unik]

btw ini semua under 50rb 🫣
```

### Key Rules
1. **Jangan hard-sell** — bungkus dalam entertainment/humor
2. **Bahasa casual** — "gue", "lo", "btw", "worth it", "gas!"
3. **Image/video WAJIB** — text-only = zero engagement
4. **Link di reply** — bukan di tweet utama
5. **Engage comments** — reply "link mana?" cepat = lebih banyak klik

### Hashtags

**Tier 1 — Selalu pakai (pilih 2-3):**
- `#ShopeeHaul` `#BarangUnik` `#ShopeeFinds` `#BarangMurah`

**Tier 2 — Rotasi:**
- `#RekomendasiShopee` `#ShopeeUnder50rb` `#BarangViral` `#ReviewJujur`

**Tier 3 — Seasonal/event:**
- `#Shopee77` (atau 88, 99, dll) → tiap sale bulanan
- `#HarbolNas` → November-Desember
- `#Gajian` → tanggal 25-1 tiap bulan
- `#THR` / `#Lebaran` → musim Ramadan

**Max 3-4 hashtags per tweet** (lebih dari itu keliatan spammy)
